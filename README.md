# ErnieBotPlugin

为AstrBot增加文心一言。

## 配置说明

| 配置项        | 说明                             | 
|------------|--------------------------------|
| API_KEY    | 千帆的API Key                     |
| SECRET_KEY | 千帆的Secret Key                  |
| APP_ID     | 文心一言的App Id                    |
| SYSTEM_MSG | 人设                             |
| MODULE     | 模型                             |
| CHANNEL_ID | 子频道号，不加引号，中间用英文逗号隔开，如[111,222] |

> 注意：文心一言模型默认联网且不可更改。如需更改欢迎fork。
